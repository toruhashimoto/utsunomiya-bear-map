# 宇都宮 クマ出没 推論マップ（非公式）

宇都宮市が公開する**クマ目撃情報**（令和8年6月8日時点・全13件）を地図化し、
各目撃の**時刻と位置**から「クマは何頭いるのか（1頭か2頭か）」を推論・可視化した、
**一市民が作成した非公式**のWebツールです。単一のHTMLファイル（`index.html`）で動作します。

👉 公開ページ（GitHub Pages）: このリポジトリの Settings → Pages を有効化すると表示されます。

## できること

- **目撃マップ**：13件の目撃地点を時系列の番号付きで地図表示（地理院タイル / OpenStreetMap）。
- **1頭仮説 ⇄ 2頭仮説の切替**：同じデータを「1頭の連続した移動経路」としても、
  「2頭（北部・中央南を動く個体／姿川・西川田に居着く個体）」としても見比べられます。
- **タイムライン再生**：6/6夕方〜6/8夜の出没を時間順にアニメーション表示。
- **新しい目撃の追加と自動再評価**：市の更新で新たな目撃が出たら入力すると、
  「既知のクマの動きで説明できるか／**別の個体**の可能性か」を移動速度から自動判定。
  （同時刻に離れた2地点で矛盾＝複数頭の疑い、を検出）入力はこの端末に保存されます。
- **更新タスク（チェックリスト）** と **安全情報・通報先**。

## 分析の結論（要約）

- 13件は **1頭のクマが夜間に市内を移動した経路**として最も無理なく説明できます（最有力・約62%）。
  全区間の所要時間と距離を計算すると、最速でも時速8.2km（明保野→大塚の7分間のみ）で、
  クマの小走りの範囲内。**同時刻に遠く離れた2地点での目撃は1件もありません**。
- ただし **2頭の可能性も残ります**（約33%）。南西の姿川・西川田に約18時間居座った密集を
  「定着した別個体」と読む構成です。
- **3頭以上は否定的**。また点の目撃データでは「母グマ＋子グマ」を1頭と区別できないため、
  実際の頭数が2頭以上の可能性は原理的に否定しきれません。
- 断定には、複数個体が同一画面に写った映像・DNA/フン鑑定・同時刻の遠隔2地点目撃のいずれかが必要です。
